# Today's Weather App

Integrated with OpenWeatherMap to get weather data by city and country name. You may get free account to retrieve API key from https://openweathermap.org/priceâ€‹.

Once you get the API key, create `.env` file under root folder, then set API key value to `REACT_APP_WEATHER_API_KEY` key.

```bash
# .env
REACT_APP_WEATHER_API_KEY={YOUR_WEATHER_API_KEY}
```
